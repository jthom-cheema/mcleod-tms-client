# OtherChargeService

This service provides methods for retrieving other charge and charge code records.

## Operations

name | role | description  
---|---|---  
[GET /otherCharges/codes](https://tms-cfaa.loadtracking.com:5690/ws/docs/services?service=OtherChargeService&operation=getChargeCodes&role=-1) |  [Users](https://tms-cfaa.loadtracking.com:5690/ws/docs/roles?role=-1) | Searches the database for charge codes matching the given request parameters.  
[GET /otherCharges/codes/{id}](https://tms-cfaa.loadtracking.com:5690/ws/docs/services?service=OtherChargeService&operation=getChargeCode&role=-1) |  [Users](https://tms-cfaa.loadtracking.com:5690/ws/docs/roles?role=-1) | Retrieves the charge code specified by the ID  
[GET /otherCharges/{id}](https://tms-cfaa.loadtracking.com:5690/ws/docs/services?service=OtherChargeService&operation=getOtherCharge&role=-1) |  [Users](https://tms-cfaa.loadtracking.com:5690/ws/docs/roles?role=-1) | Retrieves the Other Charge specified by the ID
