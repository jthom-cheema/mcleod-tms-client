# ContainerService

This service provides an operation for getting a container by ID.

## Operations

name | role | description  
---|---|---  
[GET /containers/{id}](https://tms-cfaa.loadtracking.com:5690/ws/docs/services?service=ContainerService&operation=getContainer&role=-1) |  [Users](https://tms-cfaa.loadtracking.com:5690/ws/docs/roles?role=-1) | Retrieves the RowContainer specified by the ID.
