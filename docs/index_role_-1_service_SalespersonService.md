# SalespersonService

This service provides operations for getting salesperson records and the related commission report.

## Operations

name | role | description  
---|---|---  
[GET /salespersons/{id}](https://tms-cfaa.loadtracking.com:5690/ws/docs/services?service=SalespersonService&operation=getSalesperson&role=-1) |  [Users](https://tms-cfaa.loadtracking.com:5690/ws/docs/roles?role=-1) | Retrieves the Salesperson requested by the ID  
[GET /salespersons/{id}/commissionReport](https://tms-cfaa.loadtracking.com:5690/ws/docs/services?service=SalespersonService&operation=getSalespersonCommission&role=-1) |  [Users](https://tms-cfaa.loadtracking.com:5690/ws/docs/roles?role=-1) | Produces a salesperson commission report for the salesperson ID specified.
