# DataGrafterService

This service provides access to Grafter communication services.

## Operations

name | role | description  
---|---|---  
[GET /dataGrafter/getSearchProfileParameters/{wsSearchProfile}](https://tms-cfaa.loadtracking.com:5690/ws/docs/services?service=DataGrafterService&operation=getSearchProfileParameters&role=-1) |  [Users](https://tms-cfaa.loadtracking.com:5690/ws/docs/roles?role=-1) | Retrieves a list of all parameters needed for a given WS search profile.  
[GET /dataGrafter/wsProfileSearch/{wsSearchProfileId}](https://tms-cfaa.loadtracking.com:5690/ws/docs/services?service=DataGrafterService&operation=profileSearch&role=-1) |  [Users](https://tms-cfaa.loadtracking.com:5690/ws/docs/roles?role=-1) | Allows WS search profiles to be used to define custom search content.
