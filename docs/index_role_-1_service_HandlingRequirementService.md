# HandlingRequirementService

This service provides operations that allow callers to get and search for Handling Requirements in LoadMaster LTL.

## Operations

name | role | description  
---|---|---  
[GET /handling_requirement/search](https://tms-cfaa.loadtracking.com:5690/ws/docs/services?service=HandlingRequirementService&operation=find&role=-1) |  [Users, Drivers](https://tms-cfaa.loadtracking.com:5690/ws/docs/roles?role=-1) | Finds handling requirement records with the given criteria.  
[GET /handling_requirement/{id}](https://tms-cfaa.loadtracking.com:5690/ws/docs/services?service=HandlingRequirementService&operation=get&role=-1) |  [Users, Drivers](https://tms-cfaa.loadtracking.com:5690/ws/docs/roles?role=-1) | Retrieves the handling requirement record identified by the given ID value.
