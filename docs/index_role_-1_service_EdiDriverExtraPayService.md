# EdiDriverExtraPayService

This service provides an operation for retrieving EDI driver extra pay records.

## Operations

name | role | description  
---|---|---  
[GET /ediDriverExtraPay/{id}](https://tms-cfaa.loadtracking.com:5690/ws/docs/services?service=EdiDriverExtraPayService&operation=getDriverExtraPayById&role=-1) |  [Users](https://tms-cfaa.loadtracking.com:5690/ws/docs/roles?role=-1) | Returns the requested driver extra pay record.
