# EdiOtherChargeService

This service provides an operation for retrieving EDI other charges.

## Operations

name | role | description  
---|---|---  
[GET /ediOtherCharges/{id}](https://tms-cfaa.loadtracking.com:5690/ws/docs/services?service=EdiOtherChargeService&operation=getEdiOtherCharge&role=-1) |  [Users](https://tms-cfaa.loadtracking.com:5690/ws/docs/roles?role=-1) | Retrieves an EdiOtherCharge based on the ID.
