# MileageService

This service provides an operation that calculates the distance to a stop given a GPS point.

## Operations

name | role | description  
---|---|---  
[GET /mileage/distanceToNext](https://tms-cfaa.loadtracking.com:5690/ws/docs/services?service=MileageService&operation=getDistanceToNextStop&role=-1) |  [Users, Drivers, Carriers, Carrier Drivers](https://tms-cfaa.loadtracking.com:5690/ws/docs/roles?role=-1) | This method returns the distance from the current position, in latitude/longitude, to the given stop.  
[GET /mileage/mapRoutePoints](https://tms-cfaa.loadtracking.com:5690/ws/docs/services?service=MileageService&operation=getMapRoutePoints&role=-1) |  [Users, Drivers, Carriers, Carrier Drivers](https://tms-cfaa.loadtracking.com:5690/ws/docs/roles?role=-1) |
