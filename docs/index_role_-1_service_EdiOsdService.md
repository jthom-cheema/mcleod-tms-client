# EdiOsdService

This service provides an operation for retrieving EDI OS&D records.

## Operations

name | role | description  
---|---|---  
[GET /ediOsds/{id}](https://tms-cfaa.loadtracking.com:5690/ws/docs/services?service=EdiOsdService&operation=getEdiOsd&role=-1) |  [Users](https://tms-cfaa.loadtracking.com:5690/ws/docs/roles?role=-1) | Retrieves a RowEdiOsd record based on the ID.
